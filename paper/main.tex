\title{Ghostext: Near-Distribution-Preserving LLM Steganography with Authenticated Packet Framing}

\author{Anonymous Author(s)}

\begin{document}

\begin{abstract}
We present Ghostext, a generation-based linguistic steganography protocol for large language models that lets a sender (Alice) deliver a secret message to a receiver (Bob) as natural-language cover text in the presence of a passive censor. Alice wraps her plaintext into an authenticated packet keyed by a passphrase shared with Bob, and then generates cover text token by token from a shared prompt: at each next-token step she picks one token from the model's candidate set via an arithmetic-coding-style mapping driven by the packet bits. The censor sees only this text; Ghostext is designed to preserve the model-side truncated next-token distribution up to deterministic implementation effects while the carried packet bits remain pseudorandom under authenticated encryption. Bob re-feeds the shared prompt to the same model, replays the same candidate construction at each step, and inverts the arithmetic coding to recover the packet---and from it, the secret message. We implement Ghostext end to end and report a fixed $10 \times 10$ prompt-message grid evaluation on a pinned local backend.
\end{abstract}
