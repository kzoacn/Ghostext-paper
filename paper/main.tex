\begin{abstract}
We present Ghostext, a near-distribution-preserving linguistic steganography protocol for large language models. Relative to the model's same-policy sampling distribution, the only residual deviations are deterministic integer quantization (bounded by $1/F_{\mathrm{tot}}$ per candidate) and retokenization-stability filtering; top-$p$/max-$k$ truncation is shared with the natural-cover baseline and does not count as a Ghostext-induced deformation. The protocol uses arithmetic-coding-style interval narrowing, feeds the cover generator with an AEAD-protected packet, and binds sender and receiver through a configuration fingerprint over model, tokenizer, prompt, and codec settings. We report LLM-backend evidence on Qwen/llama.cpp: an 18-trial bilingual template-known baseline and a six-case same-backend slice against implementation-aligned NLS-style single-segment baselines that isolates what payload bitstream is fed into the cover coder.
\end{abstract}
