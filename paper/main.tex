\title{Ghostext: an almost perfect stenography via large language model and arithmetic coding}

% Anonymous submission: do not include authors in this version.

\renewcommand{\shortauthors}{Anonymous Submission}

\begin{abstract}
We present Ghostext, a security-first text steganography system for a passive-censor setting with three roles: Alice, Bob, and Censor. Alice combines a shared prompt, a shared password, and a secret message, then Ghostext emits a cover text. The passive Censor only inspects the transmitted text and should not observe obvious anomalies; Bob uses the same prompt and password to recover the message. Technically, Ghostext couples encrypt-then-steganography packet construction with arithmetic-coding-style embedding over next-token distributions, plus deterministic synchronization controls (candidate policy, retokenization-stability filtering, integer quantization, finite-interval codec, and configuration fingerprint checks). The ``almost perfect'' wording in our title is a scoped design intent: under explicit assumptions, encrypted payload bits are computationally indistinguishable from random bits, and embedding is engineered to preserve model distributional behavior while consuming available entropy. We evaluate matched-configuration recoverability, fail-closed behavior under mismatch, and practical overhead with rerunnable scripts on a real local backend.
\end{abstract}
