\begin{abstract}
Ghostext is a practical linguistic steganography system that combines authenticated encryption and arithmetic-coding-style interval narrowing over large language model next-token distributions. The working target is an \emph{almost perfect} setting for a passive censor that only observes text: the embedding process should stay close to natural model behavior while preserving deterministic recovery for synchronized parties. This paper is intentionally evidence-grounded to the current implementation in \texttt{../Ghostext}. We separate implemented facts, engineering inferences, and future-work claims, and we provide a reproducible protocol-and-claims workflow designed for a CCS submission process under an explicit temporary experiment pause.
\end{abstract}
