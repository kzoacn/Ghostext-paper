\begin{abstract}
Ghostext is a practical linguistic steganography system that combines authenticated encryption with arithmetic-coding-style interval narrowing over large language model next-token distributions. The design goal is operational reliability under a passive-censor setting: deterministic sender-receiver synchronization, explicit failure semantics, and reproducible configuration binding. This paper contributes an implementation-backed protocol description, formalized bounds for key approximation effects in the current pipeline, and a concrete post-pause evaluation blueprint. We intentionally separate implemented facts from planned claims and report conservative conclusions consistent with available evidence.
\end{abstract}
