\section{Background and Threat Model}

\paragraph{Generative linguistic steganography.} In generation-based linguistic steganography~\cite{cox2005information,ziegler2019neural}, a sender maps a hidden bitstream to a token sequence by steering choices during autoregressive generation. Arithmetic coding is natural because it provides a reversible mapping between probability-partitioned intervals and bitstrings~\cite{rissanen1979arithmetic,witten1987arithmetic}. Practical systems depart from the idealized setting through candidate truncation, finite-precision quantization, and tokenization ambiguity~\cite{huang2026odstega}; Ghostext addresses these with protocol-visible deterministic rules. Unlike NLS, which converts plaintext into a bitstream by replaying arithmetic coding under a fixed context~\cite{ziegler2019neural}, Ghostext feeds the cover generator with an AEAD-protected packet so that the carried bitstream is intended to be computationally independent of plaintext semantics given the passphrase and disclosed packet length.

\paragraph{Roles and shared state.} Alice (sender), Bob (receiver), and a passive Censor see only cover text. Alice and Bob share a passphrase. They also share the protocol configuration: model/tokenizer identity, prompt, seed, candidate policy, quantization settings, and codec parameters.

\paragraph{Adversary.} The primary adversary is a passive Censor without the passphrase. Prompt knowledge is a capability axis: the released evaluation uses fixed public prompt and message sets and should be read as a template-known operational floor. Multi-message accumulation, traffic side channels, and model-family identification signals are acknowledged surfaces that require dedicated measurement beyond this paper's core evidence. Active text modification in transit and runtime compromise of sender/receiver endpoints are out of scope.

\paragraph{Assumptions.} Sender and receiver need synchronized runtime state with tokenizer-compatible behavior and identical decoding policy. Passphrase secrecy is required for packet confidentiality and integrity.
