\section{Generative AI Usage}
\label{sec:genai-usage}

Following the ACM CCS 2026 call-for-papers policy, we disclose that generative AI tools were used only as writing assistance during drafting (for language polishing and local restructuring of prose). The authors verified technical correctness, evidence alignment, and final claim wording manually against code artifacts and cited literature.

No generative AI tool was used to fabricate experimental results, alter raw measurements, or replace author judgment on security claims. All reported quantitative results are from project scripts and released result files.
