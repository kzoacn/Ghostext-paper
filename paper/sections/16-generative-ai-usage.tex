\section{Generative AI Usage}
\label{sec:genai-usage}

Following the ACM CCS 2026 call-for-papers policy, we disclose that generative AI tools were used only for language polishing and local prose restructuring during drafting. The authors verified technical correctness, evidence alignment, and final claim wording manually against code artifacts and cited literature.

No generative AI tool was used to fabricate results, alter raw measurements, or replace author judgment on security claims. All reported quantitative results come from project scripts and released result files.
